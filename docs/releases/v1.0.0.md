# Telegram-Voice-Transcriber v1.0.0

## Value Proposition
Local-first Telegram voice transcription without vendor lock-in. No Telegram Premium required.

## Core Features
- Telegram export + transcription workflow in CLI and Streamlit UI
- Local processing with faster-whisper
- Filterable collection by sender/date/type
- Resumable processing with state tracking
- Markdown export ready for notes and knowledge bases

```bash
cp .env.example .env
streamlit run app.py
```
